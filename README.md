## Electronics Projects
- MacBookPro Display Connector Replacement
  
`A friend that used to work at GeekSquad had given me a recycled MacBookPro with a broken display. Everything was working fine
except for this one key component. So what do I do? Order the $20 replacement cable and because I'm lazy and didn't want to
disassemble the whole thing, I just cut and splice. 3 years later and the display is still going strong!`

- iPad Charging Port Fix
